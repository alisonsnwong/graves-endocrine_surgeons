American National Standards Institute (ANSI) Codes for States

Source: https://www.census.gov/library/reference/code-lists/ansi/ansi-codes-for-states.html
Date: 2023-03-07



This file contains **pipe delimited** records for each state. The records are of the format:

FIPS State Code | Official United States Postal Service (USPS) Code | Name | Geographic Names Information System Identifier (GNISID)


For example: 23|ME|Maine|01779787
